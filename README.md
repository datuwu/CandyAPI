# Sweet-Inc

# How to run project

## 1. Add (or update) appsettings.json

### Example of `appsettings.json` file
      {
        "ConnectionStrings": {
          "CandyDB": "Server=(local);Database=CandyStore;Trusted_Connection=True;"
        },
        "Logging": {
          "LogLevel": {
            "Default": "Information",
            "Microsoft": "Warning",
            "Microsoft.Hosting.Lifetime": "Information"
          }
        },
        "AllowedHosts": "*"
      }

## 2. Click "IIS Express" to run API
